# 运行与配置指南

**状态：** 生效  
**最后更新：** 2026-02-06

---

## 1. 最小可用配置

必填：
- `STOCK_LIST`
- `GEMINI_API_KEY` 或 `OPENAI_API_KEY`
- 至少一个通知渠道

推荐：
- `TAVILY_API_KEYS`

---

## 2. 核心环境变量

### AI

- `GEMINI_API_KEY`
- `GEMINI_MODEL`
- `OPENAI_API_KEY`
- `OPENAI_BASE_URL`
- `OPENAI_MODEL`

### 通知

- `WECHAT_WEBHOOK_URL`
- `FEISHU_WEBHOOK_URL`
- `TELEGRAM_BOT_TOKEN`
- `TELEGRAM_CHAT_ID`
- `EMAIL_SENDER`
- `EMAIL_PASSWORD`
- `EMAIL_RECEIVERS`

### 数据与运行

- `STOCK_LIST`
- `TUSHARE_TOKEN`
- `MAX_WORKERS`
- `SCHEDULE_ENABLED`
- `SCHEDULE_TIME`
- `LOG_DIR`

---

## 3. 本地运行

```bash
python -m venv venv
pip install -r backend/requirements.txt
python -m backend.app.core.pipeline
```

---

## 4. Docker 运行

```bash
docker compose up --build
```

如需查看日志：

```bash
docker compose logs -f
```

---

## 5. 常见问题

- 推送消息截断：优先使用分批发送或文档落盘链接。
- 数据获取失败：检查数据源 Token 与网络可用性。
- Actions 未触发：检查 workflow 是否启用、Secrets 是否齐全。
