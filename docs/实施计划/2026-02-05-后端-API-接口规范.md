# 后端 API 接口规范（OpenAPI + 字段表）

**项目名称：** daily_stock_analysis  
**版本：** v1.0  
**日期：** 2026-02-05  
**状态：** 设计定稿  

---

## 1. 全局约定

- **协议**：HTTPS  
- **鉴权**：JWT（Access 1 小时，Refresh 7 天，轮换）  
- **时间格式**：ISO 8601（如 `2026-02-05T12:30:45+08:00`）  
- **分页**：`page` / `page_size`  
- **排序**：`sort=field:asc|field:desc`  
- **统一响应**：

```json
{
  "code": 0,
  "message": "ok",
  "data": {},
  "request_id": "req_abc123"
}
```

---

## 2. OpenAPI 3.1（核心接口）

> 说明：以下为 MVP 核心路径的精简版 OpenAPI 结构，字段详情见第 3 节表格。

```yaml
openapi: 3.1.0
info:
  title: daily_stock_analysis API
  version: 1.0.0
servers:
  - url: https://api.example.com
paths:
  /auth/login:
    post:
      summary: 用户登录
  /auth/refresh:
    post:
      summary: 刷新令牌（轮换）
  /auth/logout:
    post:
      summary: 退出登录
  /stocks:
    get:
      summary: 股票基础信息查询
  /stocks/sync:
    post:
      summary: 同步股票基础信息
  /watchlists:
    get:
      summary: 获取自选分组
    post:
      summary: 创建自选分组
  /watchlists/{id}:
    put:
      summary: 更新分组
    delete:
      summary: 删除分组
  /watchlists/{id}/stocks:
    post:
      summary: 分组添加股票
  /watchlists/{id}/stocks/{code}:
    delete:
      summary: 分组移除股票
  /market/list:
    get:
      summary: 行情列表（支持分组/排序）
  /stock/{code}/detail:
    get:
      summary: 个股详情
  /stock/{code}/analyze:
    post:
      summary: 触发个股分析并生成报告
  /stock/{code}/reports:
    get:
      summary: 该股历史报告列表
  /reports:
    get:
      summary: 全部历史报告
  /reports/{id}:
    get:
      summary: 报告详情（Markdown）
```

## 3. 字段表（Markdown）

### 3.1 鉴权

**POST /auth/login**
| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| username | string | 是 | 用户名 |
| password | string | 是 | 明文密码 |

**响应 data**
| 字段 | 类型 | 说明 |
|---|---|---|
| access_token | string | JWT Access |
| refresh_token | string | JWT Refresh |
| expires_in | int | Access 过期秒数 |

---

### 3.2 股票基础信息

**GET /stocks**
| 参数 | 类型 | 说明 |
|---|---|---|
| q | string | code/name 模糊搜索 |
| industry | string | 行业筛选 |
| page | int | 页码 |
| page_size | int | 每页条数 |
| sort | string | `field:asc|field:desc` |

**响应 data.items**
| 字段 | 类型 | 说明 |
|---|---|---|
| code | string | 股票代码 |
| name | string | 股票名称 |
| industry | string | 行业 |
| market | string | 市场 |

---

### 3.3 自选与分组

**GET /watchlists**
| 字段 | 类型 | 说明 |
|---|---|---|
| id | string | 分组 ID |
| name | string | 分组名称 |
| sort_order | int | 排序 |

**POST /watchlists/{id}/stocks**
| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| code | string | 是 | 股票代码 |
| sort_order | int | 否 | 排序 |

---

### 3.4 行情与个股详情

**GET /market/list**
| 参数 | 类型 | 说明 |
|---|---|---|
| watchlist_id | string | 分组过滤 |
| page | int | 页码 |
| page_size | int | 每页条数 |
| sort | string | `pct_chg:desc` 等 |

**响应 data.items**
| 字段 | 类型 | 说明 |
|---|---|---|
| code | string | 股票代码 |
| name | string | 股票名称 |
| price | number | 现价 |
| pct_chg | number | 涨跌幅 |
| volume | number | 成交量 |
| amount | number | 成交额 |
| turnover | number | 换手率 |
| volume_ratio | number | 量比 |
| net_inflow | number | 主力净流入 |
| market_cap | number | 市值 |
| industry | string | 行业 |

**GET /stock/{code}/detail**
| 字段 | 类型 | 说明 |
|---|---|---|
| latest_quote | object | 最新行情快照 |
| daily_indicators | object | 日线指标 |
| last_report | object | 最近报告摘要 |

---

### 3.5 分析与报告

**POST /stock/{code}/analyze**
| 字段 | 类型 | 必填 | 说明 |
|---|---|---|---|
| force | boolean | 否 | 是否强制重新生成 |

**响应 data**
| 字段 | 类型 | 说明 |
|---|---|---|
| report_id | string | 报告 ID |
| markdown_path | string | 报告路径 |
| generated_at | string | 生成时间 |

**GET /stock/{code}/reports**
| 参数 | 类型 | 说明 |
|---|---|---|
| page | int | 页码 |
| page_size | int | 每页条数 |
| sort | string | `generated_at:desc` |

**GET /reports/{id}**
| 字段 | 类型 | 说明 |
|---|---|---|
| markdown | string | 报告正文 |
| metadata | object | 报告元信息 |
