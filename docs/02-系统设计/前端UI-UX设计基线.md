# 前端 UI/UX 设计基线（MVP）

**状态：** 生效  
**最后更新：** 2026-02-06

---

## 1. 设计目标

- 提供 A 股场景可读性高的专业盘面体验。
- 支持桌面优先、移动端可降级浏览。
- 与后端 MVP API 对齐，优先支撑市场页、组合页、报告页。

---

## 2. 视觉与交互原则

- 深色金融风格，红涨绿跌。
- 关键数字与涨跌状态保持高对比。
- 表格密度以信息效率优先，交互控件保证可点击面积。

---

## 3. 核心页面

- 市场页：行情列表 + 分组过滤 + 分析入口。
- 组合页：分组管理 + 分组股票维护。
- 报告页：个股历史报告列表 + 报告正文。

---

## 4. 组件与规范要点

- 分组标签条（含新增分组入口）
- 行情数据表格（排序、分页、状态色）
- 报告列表与详情阅读器
- 加载、空态、错误态统一组件

---

## 5. 实施优先级

- P0：主题变量、数字与涨跌格式、市场页核心表格。
- P1：组合页分组交互、报告列表联动。
- P2：动效、无障碍和细节体验。
