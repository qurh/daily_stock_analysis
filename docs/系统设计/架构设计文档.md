### 2.2 架构特性

| 特性 | 实现方式 |
|------|----------|
| 前后端分离 | React + FastAPI |
| 微服务化 | 模块化服务设计 |
| 策略模式 | 数据源自动故障切换 |
| 事件驱动 | 盯盘告警事件 |
| RAG 增强 | 向量检索 + 知识图谱 |
| 多模型路由 | AI 路由器 |

## 4. 数据库设计

### 4.1 数据库选型

| 数据库 | 用途 | 扩展 |
|--------|------|------|
| PostgreSQL | 业务数据存储 | 主流关系型数据库 |
| pgvector | 向量索引 | RAG 语义搜索 |
| Redis（可选） | 缓存、任务队列 | 会话、实时数据 |
| 文件系统 | 文档存储 | Markdown、报告 |

### 4.2 PostgreSQL Schema 设计

#### 4.2.1 业务数据 Schema

```sql
-- 日线数据
CREATE TABLE stock_daily (
    id SERIAL PRIMARY KEY,
    code VARCHAR(10) NOT NULL,
    date DATE NOT NULL,
    open DECIMAL(10, 2),
    high DECIMAL(10, 2),
    low DECIMAL(10, 2),
    close DECIMAL(10, 2),
    volume BIGINT,
    amount DECIMAL(18, 2),
```

## 7. 安全设计

### 7.1 敏感数据加密

```python
# 使用 Fernet 对称加密
from cryptography.fernet import Fernet

class Encryptor:
    def __init__(self):
        self.key = os.getenv("ENCRYPTION_KEY").encode()
        self.fernet = Fernet(self.key)

    def encrypt(self, data: str) -> str:
        return self.fernet.encrypt(data.encode()).decode()

    def decrypt(self, encrypted: str) -> str:
        return self.fernet.decrypt(encrypted.encode()).decode()
```

### 7.2 API Key 存储

- API Key 加密后存储在 `cfg_ai_models.api_key_encrypted`
- 运行时动态解密
- 日志中不输出 Key
