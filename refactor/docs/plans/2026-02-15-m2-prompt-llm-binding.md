# M2 Prompt LLM Binding Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Bind chat generation to Prompt Center active version and route final generation through pluggable LLM provider.

**Architecture:** Extend `ChatService` to resolve active prompt template (`prompt.chat.reply`) from `PromptService`, render with runtime variables, then generate response via `LLMProvider` abstraction. Keep API contracts unchanged while adding generation trace (`prompt_ref`, `llm_provider`, `llm_model`) in assistant message metadata.

**Tech Stack:** Python 3.10+, FastAPI, sqlite3, pytest.
