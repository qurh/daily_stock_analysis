1. 风险描述：assistant 回复目前为规则生成，不具备真实大模型推理能力。
2. 影响范围：回答质量与自然语言表现仍有限。
3. 缓解措施：下一阶段接入可替换 LLM provider 与 Prompt Center 绑定。

## 8. 关键决策

1. 决策内容：M2 第二阶段优先交付“可验证 API 闭环”，推迟模型能力增强。
2. 决策原因：先建立持久化数据链路和回归测试基线，降低后续集成风险。
3. 影响模块：Interaction Gateway、Memory Service、Knowledge Service、Prompt Center（后续联动）。
